# ClassSync Pro - ローカル版の使い方

## 🚀 超簡単！サーバー不要で動作

### ダウンロード方法

1. **Code**ボタン → **Download ZIP** をクリック
2. ZIPファイルを解凍
3. **`npm install`を実行**（初回のみ）
4. **`npm run build`を実行**してdistフォルダを生成
5. `dist`フォルダ内の**index.html**をダブルクリック

---

## 📱 ワンクリック起動

### distフォルダだけで動作！

1. `npm run build` を実行してdistフォルダを生成
2. **distフォルダをコピー**してどこでも使える
3. `dist/index.html`をダブルクリックするだけ！

### デスクトップショートカットを作成

`dist/index.html`のショートカットをデスクトップに作成すれば、いつでもワンクリックで起動！

---

## 💾 データについて

- データはブラウザのlocalStorageに保存されます
- **file:///** プロトコルでも動作します（サーバー不要）
- 定期的にエクスポート機能でバックアップしてください

`dist`フォルダをZIPにして配布すれば、誰でも解凍してHTMLを開くだけで使えます！
